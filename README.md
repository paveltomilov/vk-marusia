# 🎬 VK Marusia

> Интерактивный каталог фильмов с авторизацией, фильтрацией по жанрам и детальными карточками. Проект создан для изучения React Router, API интеграций и SPA архитектуры.

## Особенности

- **Авторизация** — модальное окно с формой входа
- **Жанры** — фильтрация фильмов по категориям
- **Детальная карточка** — информация о фильме (описание, рейтинг, постер)
- **SPA маршрутизация** — плавная навигация без перезагрузки страницы
- **Адаптивный дизайн** — корректное отображение на всех устройствах
- **Ленивая загрузка** — компоненты подгружаются по мере необходимости

---

## Технологии

| Технология          | Назначение         |
| ------------------- | ------------------ |
| **React 18**        | UI библиотека      |
| **React Router v6** | Маршрутизация      |
| **Vite**            | Сборка проекта     |
| **SCSS Modules**    | Стилизация         |
| **Axios**           | HTTP запросы к API |

## Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/paveltomilov/vk-marusia.git
cd vk-marusia

# Установить зависимости
npm install

# Запустить dev сервер
npm run dev
```
